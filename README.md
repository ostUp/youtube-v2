# About

The project was created with:

Typescript, React.js, React Router, Redux Toolkit, Tailwind CSS, Axios and YouTube API


## Run Locally

Fork the repo\
Clone the repo

```bash
  git clone https://link-to-account/youtube-v2
```

Go to the project directory

```bash
  cd youtube-v2
```

Install dependencies

```bash
  npm i
```

Start the project

```bash
  npm start
```
